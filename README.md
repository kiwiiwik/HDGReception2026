\# HDGReception2026



\## Overview

Incoming calls arrive via Twilio and are answered by an ElevenLabs 

conversational AI agent acting as a receptionist.



The agent determines who the caller wants to speak with, then passes 

that information to an Azure Web App. The Azure app initiates an 

outbound call via Twilio to connect the caller.
